Raster App Changelog

2.0.2 Sept 27, 2016

  • Fix bug causing GeoPDF links to download the corresponding web page (#16).

2.0.1 Sept 19, 2016

  • Fix bug preventing "cat" & "catGroup" URL parameters from working.

2.0.0 Sept 19, 2016

  • Fix issues with preview and extent buttons (#13).
  • Force (most) browsers to download all files instead of opening them (#14).
  • Add a spinner to the download button (#11).
  • Support Web Mercator previews using the ServiceName field (#2).
  • Support tile indices with overlapping polygons (#5).
  • Show project-based metadata and reports only once in the side bar (#3).
  • Removed "ArcGIS Layer File" link from more info dialog's (#4).
  • Better layout for title in result panel (#9).
  • Tweaks to help panel (#10).
  • Switch to web mercator (#7).
  • Upgraded all dependencies including Dojo & ESRI JS API.
  • Ported most layouts from Dijit to Bootstrap.
  • Upgraded project to latest boilerplate (#6).

1.4.1 Aug 5, 2013

  • Fixed bug that was showing extra results when using cat or catGroup url parameters. Ref #24833.

1.4.0 May 1, 2013

  • Fixed bug with 10.1 and layer definitions that was preventing ?cat=... url's from working.
  • Pushed to 10.1 servers.

1.3.0 Nov 14, 2012

  • .met and .jpq file types are now forced to download.
  • Search button was removed and the app now automatically begins the search when a graphic is drawn or an address is found.
  • Better chronological sorting of search results.
  • When accessing the app using the "cat" or "catGroup" url parameters, a link to the main app is displayed in the Step 1 and Step 3 panes.
  • Added a background color to the search result groups to make it more apparent that they are click-able.
  • The "Selected Tiles" drop down in the Downloads pane now defaults to be open.
  • The area of interest graphics now show on the preview map.

1.2.0 Aug 14, 2012

  • The main product categories (Aerial Photography, Contours, etc.) are now collapsible in the Results pane.
  • Added catGroup and title url parameters to allow groups of products to be specified.
  • Added the back and full extent buttons to the preview map.
  • Added a disclaimer to the Download pane for Auto-Correlated DEM products only.
  • Moved the toolbox to a sidebar layout.

1.1.1 Aug 2, 2012

  • Fixed bug that caused "Getting tiles..." message to show up incorrectly in the Download pane.
  • All download links open in new tabs to prevent loss of the application in the current tab.
  • Switched backend map service to use the new SHOW field instead of searching for 'n/a' values for *_Extents feature classes.
  • Shrunk the toolbox window height by 50 pixels to hopefully prevent scrolling issues.
  • Change the query for tiles to download to use the intersection of the area of interest and the extent feature instead of just the area of interest alone. This fixed a few bugs including incorrect FTP paths and to many tiles showing in download tab.
  • Changed the link to download tiles in the map popup from "Image Tile" to "Tile" to eliminate confusion with non-aerial photography tiles.
  • Added "Flight Date Location" to the more info box for aerial photography products.
  • Removed "Resolution" from Topo more info box and added "Year" to LiDAR more info box.

1.1.0 Jul 18, 2012

  • Added Topo maps (DRG's).
  • Added additional text to the feedback form.

1.0.1 Jul 18, 2012

  • Fixed IE8 bug related to the preview buttons.
  • Made clear preview button visible even when the aerials are not selected.
  • Hid clear preview button when no results are found.
  • Fixed LiDAR previews.
  • The map now shows all features when a category is passed via the url instead of just the first feature found.
  • Fixed a bug that was causing the "No data found" text to show up even if there were results found.
  • Added a filter to remove {}'s in categories passed in via the url.
  • Clear drawings button now disables any drawing tools that are activated.

1.0.0 Jul 17, 2012

  • Added a link to the associated layer file for products that have layer files.
  • Fixed bug that was causing the extent of the map to not stay consistent when switching between different preview layers.
  • Changed links in popup window from file extentions to 'Image Tile', 'World File', & 'Metadata'.
  • Fixed bug in Chrome that was causing the drawing toolbar buttons to resize on hover.
  • Added preview buttons for product types other than aerial photography.

0.5.0 Jul 2, 2012

  • Mb's that are over 1,000 are now converted to Gb's.
  • Fixed a bug that caused the app to crash if a new search was performed while the preview map was up. Now when you click search or click on a draw tool the app automatically switches back to the default map if the preview map is visible
  • Changed image service layers to Bilinear interpolation.
  • Hid all graphics on the preview map. This may be changed to show only the outlines in a future version.
  • Fixed the "Extent" buttons.
  • Fixed nasty layout bug in IE9 related to the product result boxes.
  • Added a scale bar to the preview map.
  • Added Rick's email to get feedback.
  • Added the ability to lock the app into a specific data product by passing it in as a url parameter.
  • Added a link to the image server page in the help pane.

0.4.0 Jun 27, 2012

  • The map now switches to infinite zoom when previewing a product.
  • The tiles on the map and pop-up window are now cleared when you define a new area of interest or do a new search.
  • Added a 'Help' pane to the Toolbox with map navigation help.

0.3.0 Jun 21, 2012

  • Tiles are now shown on the map (in blue) when you click on the download button.
  • Clicking on a tile on the map shows a popup with related downloads.
  • 'Null' values are now appropriately handled for the METADATA and REPORT fields in the DEM and LiDAR *_Extents feature classes.
  • Fixed bug where the drawing toolbar was not displaying correctly in IE or Firefox.
  • Added a feedback button.
  • Added a back button.
  • Shortened the title to allow for better layout on smaller screens.

0.2.0 Jun 18, 2012

  • Fixed bug where a drawing tool could not be unselected.
  • Added files associated with the tile to the download section. For example .xml or .tfw.
  • Added button to drawing toolbar for clearing the area of interest graphics.
  • Added a max-height and scrollbar to the download pane to help with long lists of tiles.
  • Worked with ftp server admin to force the browser to download all file types instead of trying to open them up.

0.1.0 Jun 5, 2012

  • Added release notes page.
  • Added download pane functionality including listing selected tiles and wiring them to the map.
  • Toned down the generalization on the return extent graphics (yellow).
  • Handled IN_House = 'NO'.
  • Added 'more info' link and associated dialog to product result widget.
